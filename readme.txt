This plugin allows you to embed a Twitch stream and chat in a responsive layout with live status detection, caching, and theme toggle.

== Description ==
Easily embed a Twitch stream and chat on any page or post using a shortcode. Automatically shows "Live" or "Offline" status with color indicators.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin via the Plugins menu.
3. Configure under "Settings > Stream Embed for Twitch".
